express-app

A simple Node.js + Express application built as a Docker + CI/CD pipeline example

<h3>Purpose</h3>

This project was created as part of a Docker and CI/CD course to demonstrate:

how to containerize a Node.js + Express application,

how to build, test, and deploy it via an automated pipeline (e.g., using GitHub Actions, GitLab CI, etc.),

basic infrastructure practices such as environment configuration, building and deploying as a container image.

Access with browser http://localhost:8080
